# Char Library

The `(scheme char)` library provides the procedures for dealing with characters.

For more information see the [R<sup>7</sup>RS Scheme Specification](../../r7rs.pdf).

- `char-alphabetic?`
- `char-downcase`
- `char-lower-case?`
- `char-numeric?`
- `char-upcase`
- `char-upper-case?`
- `char-whitespace?`
- `digit-value`
- `string-upcase`
- `string-downcase`
